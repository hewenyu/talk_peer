# talk_peer
这是一个基于rust 的练手项目


基于区块链通信的通讯软件可以采用去中心化的设计，使用户之间可以直接通讯，而不必经过第三方的服务器。以下是可能的功能大纲：


* 用户注册和身份验证
    * 允许用户创建账户并记录在区块链上
    * 要求用户提供必要的身份验证信息以(邮件的形式)，以确保他们是真实的用户
* 消息传递和接收
    * 允许用户在区块链上发送和接收消息
    * 将消息和用户信息记录在区块链上，以确保信息安全和完整性
    * 使用加密算法对消息进行加密，以保护隐私
* 好友管理
    * 允许用户添加好友并建立联系人列表
    * 使用区块链记录好友关系，以确保安全性和隐私保护
    * 允许用户对不受欢迎的用户进行屏蔽或拉黑
* 通讯记录和存档
    * 记录用户的通讯历史，以便他们可以查看以前的消息
    * 使用区块链存储历史消息，以确保数据的完整性和安全性
    * 允许用户搜索和导出通讯记录
